 Instructions:
  1. Drag and drop contents of dxvk-patch into .trex folder in Portal RTX bin
  2. Drag and drop one of the rtx.conf presets into Portal RTX directory, make sure to replace existing rtx.conf

 Tested GPUs and resolutions
  ARC A770 at 1440p 30fps (Medium Preset)

 FSR 3.1 can be toggled through ALT + X and changing upscaler to DLSS

 Warning: While in ALT + X dev menu, do not go to the remix graphics menu, this will revert some changes and reduce performance (Only if using TAA-U or NIS, Does not affect FSR)
